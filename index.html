Projects Participated

NERDSS (Nonequilibrium Reaction-Diffusion Self-assembly Simulator) - A structure-resolved reaction-diffusion simulation software developed by Johnson Lab, JHU
ioNERDSS - A Python library that provides user‐friendly tools for setting up and analyzing output from the NERDSS reaction‐diffusion simulator. I participated in the implementation of automatic setup of a coarse-grained model, a self-assembly reaction network, an ODE solution, and a Gillespie simulation from any PDB code.
SLIMED (Subdivision Limit Membrane Energetics and Dynamics Model) - A membrane energetic and dynamics model based on Loop subdivision method (in dev, currently private repo)
Rosetta - Computational modeling and analysis of proteins and other biomolecules. I worked on the combinatorial expansion of the carbohydrate database.

Personal Side Projects

Reaction IVP solver - Provides a set of functions for working with chemical reaction systems, including parsing reaction strings, solving reaction kinetics, and modifying reaction rates.
Complex Scheduling - A quick example of how to schedule a set of tasks with precedence constraints across multiple bins (e.g., workers or machines), using the OR-Tools CP-SAT solver. The goal is to minimize the overall makespan — the total time required to complete all tasks.
FFT example - A quick example for teaching students the concept and usgae of FFT module from scipy.
